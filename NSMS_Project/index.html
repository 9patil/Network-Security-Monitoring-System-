Network Security Monitoring System (NSMS)

Project by Pratik Patil (9patil)

About Project

This is a simple Network Security Monitoring System built using Python and Nmap.

Features:

  • Detects active devices
  • Detects unauthorized MAC addresses
  • Finds open ports
  • Generates logs

Project Files

  • nsms_scan.py – Main script
  • authorized_devices.txt – Add trusted devices
  • run.bat – Double-click run
